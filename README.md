# SF Portfolio

Personal developer portfolio built with React 19, featuring multi-language support, dark/light theme switching, and a contact form integrated with a backend API.

## Live Demo

[sf-portfolio-alpha.vercel.app](https://sf-portfolio-alpha.vercel.app)

## Features

- Multi-language support (TR/EN) via Redux
- Dark / Light theme toggle
- Hire Me modal with contact form
- Projects, Skills, and Profile sections
- E2E tests with Cypress

## Tech Stack

| Category | Technologies |
|----------|-------------|
| Framework | React 19 + Vite |
| State | Redux Toolkit |
| Data Fetching | TanStack React Query v5 |
| Styling | Tailwind CSS v4 |
| HTTP | Axios |
| Testing | Cypress |

## Getting Started

```bash
npm install
npm run dev
```

Run E2E tests:

```bash
npm run cypress:open
```
